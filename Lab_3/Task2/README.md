# Клиент-серверное приложение (Задание №2)

Данная программа реализует многопоточную архитектуру "Клиент-Сервер" на языке C++ с использованием стандартов C++20.

## Описание
- **Сервер:** пул потоков-воркеров, обрабатывающих очередь задач. Результаты хранятся в `std::unordered_map` в виде `std::shared_future`.
- **Клиенты:** 3 независимых потока, генерирующих по N случайных задач (Sin, Sqrt, Pow).
- **Синхронизация:** используются `std::promise`, `std::future`, `std::mutex` и `std::condition_variable`.

## Сборка и запуск

1. Перейдите в директорию задания:
   ```bash
   cd Lab_3/Task2/
   ```

2. Скомпилируйте проект:
   ```bash
   mkdir -p build && cd build
   cmake ..
   make
   ```

3. Запустите приложение:
   ```bash
   ./client_server_app
   ```

## Результаты
После выполнения программы:
- В консоль выведется время выполнения и статус тестов (SUCCESS).
- В файле `res.txt` будут сохранены все вычисленные значения в формате:
  `Type Argument = Result id = ID`
